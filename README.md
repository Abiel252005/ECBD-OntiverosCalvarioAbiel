## Descripción general

Este repositorio contiene las prácticas realizadas durante el primer parcial de la materia **Extracción del Conocimiento de Bases de Datos**.

El propósito del repositorio es presentar de forma organizada los notebooks, datasets y documentación desarrollada durante el curso. Las prácticas integran procesos de carga, exploración, limpieza, análisis descriptivo, visualización e interpretación de datos, con el objetivo de transformar datos en información útil.

El repositorio también sirve como evidencia del uso de GitHub para mantener una estructura clara, registrar avances mediante commits y documentar correctamente el trabajo realizado.

## Instrucciones generales de ejecución

Para utilizar el repositorio de forma local, se recomienda clonar el proyecto, instalar las dependencias necesarias y abrir los notebooks desde Jupyter Notebook.

```bash
git clone https://github.com/Abiel252005/ECBD-OntiverosCalvarioAbiel.git
cd ECBD-OntiverosCalvarioAbiel
pip install pandas numpy matplotlib jupyter
jupyter notebook
```

Dentro de Jupyter Notebook se debe ingresar a la carpeta `Notebooks` y abrir el laboratorio que se desea revisar o ejecutar.

Para comprobar que un notebook funciona correctamente, se recomienda ejecutar todas las celdas desde el inicio utilizando la opción:

```text
Kernel → Restart Kernel and Run All Cells
```

## Uso de rutas relativas

Los notebooks se encuentran dentro de la carpeta `Notebooks`, mientras que los datasets están dentro de `DataSet`.

Por esa razón, al cargar archivos desde un notebook se debe utilizar la ruta relativa:

```python
import pandas as pd

df = pd.read_csv("../DataSet/nombre_del_archivo.csv")
```

Ejemplo:

```python
netflix = pd.read_csv("../DataSet/netflix_titles.csv")
facturas = pd.read_csv("../DataSet/ventas-por-factura.csv")
```

## Interpretación de resultados

La interpretación es una parte esencial del repositorio, ya que no se evalúa únicamente el código, sino también la capacidad para explicar los resultados obtenidos.

Cada notebook debe incluir explicaciones que respondan preguntas como:

```text
¿Qué muestra la gráfica?
¿Qué variable tiene mayor frecuencia?
¿Qué tendencia se observa?
¿Qué valores parecen atípicos?
¿Qué relación existe entre las variables?
¿Qué conocimiento se obtiene?
¿Qué conclusión puede redactarse con base en el análisis?
```
